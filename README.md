# 智慧党建信息管理平台

## 项目简介
基于Vue3 + Vite构建的智慧党建信息管理平台，提供党员档案管理、培训信息、组织关系等功能。

## 功能特性
- 🏛️ 党员档案管理（支持Excel导入导出）
- 📚 培训信息管理
- 🏢 组织关系展示
- 👥 角色权限管理
- 📊 数据统计展示

## 技术栈
- Vue 3
- Vue Router 4
- Vite
- XLSX (Excel处理)

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 浏览器访问
打开浏览器访问：`http://localhost:5173/`

## 登录方式

### 党校校长（查看全部）
- 账号：`admin`
- 权限：可以查看所有党支部的信息

### 书记/副书记（查看本年级）
- 账号：`2021`、`2022`、`2023`、`2024`、`研究生`
- 权限：只能查看本年级的党支部信息

## 页面说明

### 首页大屏
- 党校校长：显示全校统计数据
- 书记/副书记：显示本年级统计数据

### 党员档案
- 支持Excel导入导出
- 党员基本信息管理
- 发展过程跟踪

### 培训信息
- 培训记录管理
- 考核成绩统计
- 开始时间记录

### 组织关系
- 党支部结构展示
- 年级分组管理
- 成员信息查看

## 项目结构
```
src/
├── views/
│   ├── Login.vue          # 登录页面
│   ├── Dashboard.vue      # 首页大屏
│   ├── Development.vue    # 党员档案
│   ├── Training.vue       # 培训信息
│   └── Organization.vue   # 组织关系
├── router.js              # 路由配置
├── main.js               # 应用入口
└── App.vue               # 根组件
```

## 开发说明
- 使用Vue 3 Composition API
- 响应式设计，支持多种屏幕尺寸
- 现代化UI设计，美观易用
